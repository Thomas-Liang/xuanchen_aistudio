# XuanChen AI Studio 功能说明

> 基于 Gemini 架构的专业 AI 图像生成工作室

---

## 🎨 核心功能

### AI 图像生成
- **文本生成图像**: 输入描述词，AI 自动生成图像
- **图像编辑**: 上传参考图，结合文字描述进行编辑
- **宽高比控制**: 支持 1:1, 2:3, 3:2, 4:3, 16:9 等多种比例
- **分辨率选择**: 1K / 2K / 4K 三档输出质量

### 多模型支持
| 模型 | 特点 | API 类型 |
|------|------|----------|
| Banana (Nano) | 轻量快速，适合构图验证 | 代理节点 |
| Banana Pro | 语义理解强，细节丰富 | 代理节点 |
| Gemini 3 Pro | 谷歌官方最新模型 | 官方 API |
